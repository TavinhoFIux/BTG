Angular Angular Framework

Angular is an open-source, front-end, TypeScript-based web application platform led by the Angular Team at Google and a community of individuals and businesses. Angular is a complete rewrite of AngularJS, done by the same team that helped it.

.Net Core .Net Core Framework

.NET is a free and open source framework for Windows, Linux and macOS operating systems. It is an open source successor to the .NET Framework. The project is primarily developed by Microsoft and released under the MIT License.
